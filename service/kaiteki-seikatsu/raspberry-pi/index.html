ラズベリーパイの活用

Raspberry Pi（ラズベリーパイ）をご存知でしょうか。ラズベリーパイは英国のラズベリーパイ財団によって開発され、世界的に人気を博している小型のマイコンボードです。

Raspberry Pi – Teach, Learn, and Make with Raspberry Pi

Raspberry Pi 3 Model B

モニター、キーボード、マウスなどを接続することでパソコンのように使用することもできる他、汎用の入出力ポート（GPIO）を使用した電子工作も盛んです。

当社ではこのラズベリーパイを活用して住まいをＩＴ化し、生活を便利に快適にする様々なアイデアを考案し、みなさまにご提案したいと考えています。
